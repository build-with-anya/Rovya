#Rovya

Balance every role. Without losing yourself.
Rovya is a mental wellness and life management app designed for women juggling multiple roles—career, family, and personal life.

🧠 Problem
Women today face:
Mental overload
Burnout
Guilt from balancing multiple responsibilities

Most solutions don’t reflect real-life complexity.

💡 Solution

Rovya helps users:

Track mental load across roles
Understand imbalance
Get simple, daily guidance

🚀 Core Features
🔄 Daily Sync – quick mood & stress check-in
🗓️ Today’s Flow – energy-based daily planner
⚡ Quick Reset – 2–5 min stress relief tools
📊 Insights – balance score & trends

🧩 How It Works
User checks in
Rovya analyzes mood + roles
Generates plan & insights
User improves balance over time

💎 Value
Rovya is not just a wellness app—it’s a life balance system.

🧭 Roadmap
MVP: Sync, Planner, Reset, Insights
Next: AI support, community, coaching
